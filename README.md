# Taller 2 — Framework Kedro

Material del taller "De scripts en Python a pipelines reproducibles con Kedro",
continuación del Taller 1. Sigue el tutorial oficial de spaceflights de Kedro
(kedro.org) en cinco módulos (0 a 4).

## Qué hay en este repositorio

- **`material_taller/Taller_Kedro_Completo.docx`** — la guía completa del
  taller: Introducción, Módulos 0 a 4, y Cierre.
- **`material_taller/material_taller/modulo_1` a `modulo_4`** — archivos de
  referencia por módulo (catálogo, pipelines, tests, hooks, namespaces,
  dataset factories y la plantilla de entrega). Son material de respaldo
  para comparar, no para copiar y pegar de una vez — la guía explica en cada
  módulo qué escribir tú mismo.

## Cómo usarlo

Clónalo como parte del Módulo 0 de la guía (sección 0.2):

```powershell
cd C:\
mkdir kedro
cd kedro
git clone https://github.com/yacastanot/taller-2-kedro.git
```

Este repositorio es de solo lectura para ti — es el material del taller, no
tu entrega. Tu propio proyecto Kedro (el que vas a construir siguiendo la
guía) va en una carpeta aparte, con su propio repositorio nuevo, tal como se
explica en el Módulo 3.

## Requisitos previos

Haber completado el Taller 1 (Python, entorno virtual, Git/GitHub, IA en el
IDE). La guía no vuelve a explicar esos fundamentos.
